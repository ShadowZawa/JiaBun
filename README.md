# JiaBun
JiaBun (甲奔) 是一款以「虛擬人設互動」為核心的智慧助理 App，結合 AI 聊天、情感陪伴與生活管理工具，專為學生與年輕使用者設計，目標是減輕日常壓力、改善生活習慣並提升幸福感。

## 核心理念
- 用有溫度的互動陪伴，讓使用者感到被理解與支持。  
- 將 AI 能力與實用的生活管理功能結合，協助使用者提高生活品質與學習效率。  
- 注重隱私與安全，提供透明的資料使用說明（請依實際情況補充）。

## 主要功能
- AI 虛擬人設互動：可自訂或選擇不同個性的虛擬角色進行對話與互動。  
- 情緒支持與陪伴：傾聽、回應情緒、提供放鬆建議或短期情緒調節技巧。  

## 預計新增 (開發中)
- 生活管理：待辦事項、行事曆、提醒、習慣追蹤與番茄鐘等工具。  
- 學習助理：筆記整理、複習提醒、學習計畫建議與時間管理支援。  
- 個人化設定：依使用習慣與偏好調整提醒頻率、角色風格與語氣。
- 理財規劃：為使用者訂製專屬的理財規劃
- 日常關心：在使用者需要時自動跳出彈窗提示進行關心
- 自訂義位置：使用者可以在需要時透過輸入地址來設定搜尋的中心點
