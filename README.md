# JiaBun
JiaBun (甲奔) 是一款以「虛擬人設互動」為核心的智慧助理 App，結合 AI 聊天、情感陪伴與生活管理工具，專為學生與年輕使用者設計，目標是減輕日常壓力、改善生活習慣並提升幸福感。

## 核心理念
- 用有溫度的互動陪伴，讓使用者感到被理解與支持。  
- 將 AI 能力與實用的生活管理功能結合，協助使用者提高生活品質與學習效率。  
- 注重隱私與安全，提供透明的資料使用說明（請依實際情況補充）。

## 主要功能
- AI 虛擬人設互動：可自訂或選擇不同個性的虛擬角色進行對話與互動。  
- 情緒支持與陪伴：傾聽、回應情緒、提供放鬆建議或短期情緒調節技巧。  

## 有什麼用？
- 減少日常壓力：用對話與互動提供即時情緒抒發與建議。  
- 幫助建立學習節奏：結合提醒與學習規劃提升專注與效率。  
- 提供穩定的陪伴感：在忙碌或孤單時，有虛擬對象可傾訴與互動。

## 預計新增
- 生活管理：待辦事項、行事曆、提醒、習慣追蹤與番茄鐘等工具。  
- 學習助理：筆記整理、複習提醒、學習計畫建議與時間管理支援。  
- 個人化設定：依使用習慣與偏好調整提醒頻率、角色風格與語氣。
- 理財規劃：為使用者訂製專屬的理財規劃
- 日常關心：在使用者需要時自動跳出彈窗提示進行關心

## 隱私與資料使用
JiaBun 尊重使用者隱私，以下條款說明我們蒐集哪些資料、用途、保護方式，以及使用者的權利。若本產品有法律、合規或技術上的更動，我們會在 App 或隱私政策頁面更新並註明生效日。

1. 我們會收集哪些資料
   - 帳號資料：註冊資訊（如電子郵件、使用者名稱）及設定偏好。  
   - 對話與互動內容：使用者與虛擬人設的對話內容（包含文字、選擇題回覆等），在取得使用者同意的情況下收集以用於改進模型與服務品質。  
   - 使用與裝置資訊：裝置型號、作業系統版本、App 版本、使用時長與操作紀錄（用於分析與偵錯）。  
   - 故障與日誌資料：當 App 異常時，我們會收集錯誤日誌以利除錯。  
   - 其他：若使用某些功能（例如上傳檔案、語音）會另行說明該功能的資料處理方式。

## 常見問題（FAQ）
- Q：JiaBun 會把我的對話外流嗎？  
  A：請參閱隱私政策；系統設計上會盡可能保護使用者資料並提供選項讓使用者控制分享設定，僅我們的後台與Google Gemini模型會擁有你的對話資料。  
- Q：可以同時建立多個虛擬人設嗎？  
  A：是，使用者可新增與切換不同角色 (Under Development)
